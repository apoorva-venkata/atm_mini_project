# ATM Mini Project

A simple and interactive ATM simulator built using HTML, CSS, and JavaScript. Users can simulate entering a card name, verifying a password, checking account balance, and withdrawing virtual money. Perfect for beginners to practice JavaScript logic and front-end development.

## Features
- Card name validation
- Password verification
- Balance check
- Withdraw virtual money
- Simple, user-friendly UI

## Technologies Used
HTML | CSS | JavaScript
